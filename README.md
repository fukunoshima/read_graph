# read_graph

## 使い方

settingを編集

```
# setting ----------------------
filename = "" # filename

xmin = 0      # ranges of graph
xmax = 20
ymin = 0 
ymax = 4 
# ------------------------------
```

filename: 値を読みたい図へのpath
xmax, xmin: 値を読みたい図のx軸の最大・最小値
ymax, ymin: 値を読みたい図のy軸の最大・最小値

図はscreen shotするとき、x軸の最大・最小値をうまく揃えると読み取りやすいと思います。
